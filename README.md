# Bistro_buddy-Food-ordering-application


 Bistro Buddy 🍽️

Bistro Buddy is a full-stack food ordering web application built using **React.js, Spring Boot, and MySQL**.  
It allows customers to browse menus, place orders, book tables, and view order history, while admins can manage menu items and monitor customer orders.


a. Features

1. Customer Features
- User Registration & Login Authentication
- Browse Food Menu
- Add Items to Cart
- Place Orders
- Table Booking
- View Order History
- Responsive UI

2. Admin Features
- Admin Login
- Add Menu Items
- Update Menu Items
- Delete Menu Items
- View Customer Orders
- Manage Restaurant Menu
